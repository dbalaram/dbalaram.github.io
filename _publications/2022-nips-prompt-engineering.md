Fine-grained fashion attribute classification (e.g., sleeve styles, trending silhouettes) is challenging in e-commerce due to the cost of labeled data and the granularity required. This work explores how expert-crafted, domain-specific prompts significantly improve zero-shot classification accuracy with CLIP, compared to coarse or naive alternatives. The key insight is that detailed textual descriptions create better-defined embedding spaces — enabling precise attribute recognition without any task-specific training data.
